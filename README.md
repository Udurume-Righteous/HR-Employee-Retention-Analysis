# HR Employee Retention Analysis

## Project Overview
This project analyzes HR employee data to determine factors affecting employee turnover.

## Tasks Performed
- Exploratory Data Analysis (EDA)
- Correlation Analysis
- Salary vs Retention Visualization
- Department vs Retention Visualization
- Logistic Regression Model
- Model Accuracy Evaluation

## Key Insights
- Satisfaction level has the strongest impact on retention.
- Low salary employees are more likely to leave.
- Certain departments have higher turnover rates.

## Model Performance
Logistic Regression model achieved approximately 83% accuracy.

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-Learn
